## TEA-HUB - Frontend

Instale as dependências

```bash
npm install

```

<br/>

Rode a tela
```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Abra [http://localhost:3000/login](http://localhost:3000/login) no browser para ver a tela.
